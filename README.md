# Workforce Manager

A modern **Employee Task Management System** built with **React, Vite, Tailwind CSS, and Local Storage**. This application allows administrators to assign tasks to employees while enabling employees to manage and update the status of their assigned work.

---

##  About the Project

Workforce Manager is a responsive task management application designed to simplify communication between administrators and employees.

The Admin Dashboard allows administrators to assign tasks to different employees, while the Employee Dashboard enables employees to track their assigned work and update task statuses such as **New**, **Active**, **Completed**, and **Failed**.

This project was built to strengthen my understanding of React concepts including component-based architecture, Context API, state management, props, conditional rendering, and browser Local Storage.

---

##  Features

###  Admin Dashboard

- Secure Admin Login
- Create and Assign Tasks
- Assign Tasks to Individual Employees
- View Employee Task Statistics
- Track Completed, Active, Failed and New Tasks

###  Employee Dashboard

- Secure Employee Login
- View Assigned Tasks
- Accept New Tasks
- Mark Tasks as Completed
- Mark Tasks as Failed
- Live Task Count Updates

###  General Features

- Responsive UI
- Modern Dashboard Design
- React Context API
- Local Storage Data Persistence
- Component-Based Architecture
- Dynamic Task Management
- Fast Performance using Vite

##  Installation

Clone the repository

```bash
git clone https://github.com/your-username/Workforce-Manager.git
```

Go into the project directory

```bash
cd Workforce-Manager
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```
